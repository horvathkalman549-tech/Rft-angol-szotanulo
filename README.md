# RFT Angol Szótanuló

## Projekt célja

Egy webes angol szótanuló alkalmazás létrehozása.

A felhasználó:
- B1 szintet választhat
- B2 szintet választhat
- C1 szintet választhat
- szavakat gyakorolhat

- HTML

- JavaScript
